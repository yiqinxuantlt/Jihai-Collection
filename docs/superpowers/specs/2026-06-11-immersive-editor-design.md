# 沉浸写作舱编辑器升级设计

## 背景

当前网站已经完成“书架 → 书籍详情列表 → 单独编辑页”的主流程，编辑页也具备富文本输入和选区悬浮工具条。下一步优化聚焦在创作体验：让编辑页更安静、更适合长时间写随笔，同时保留必要的书籍元信息和保存/导出能力。

## 目标

- 让编辑页视觉重心更集中在写作纸面上。
- 增加专注模式，临时隐藏侧栏和次要操作。
- 右侧信息区升级为创作侧栏，优先呈现字数、书籍、更新时间和写作提示。
- 悬浮工具条更轻、更贴近选区，减少打扰。
- 不改变书架和书籍详情列表的主流程。

## 设计

编辑页保留顶部细栏。左侧为“返回记录列表”和当前位置，右侧为保存状态、专注模式、预览、导出、删除和主题切换。进入专注模式后，隐藏预览、导出、删除、创作侧栏，纸面居中并扩大阅读宽度；再次点击“退出专注”恢复完整编辑页。

写作纸面增加更稳定的最大宽度和更舒适的内边距。摘录区继续保持轻引用视觉，正文区增加最小高度和更自然的段落排版，让长文输入时不显得拥挤。

右侧创作侧栏顶部显示创作状态，包括当前字数、关联书籍、最近更新日期；下方仍提供记录类型、书籍、页码、主题、标签等元信息编辑；底部保留写作提示。

## 验收

- 点击记录进入编辑页后，页面更聚焦纸面。
- 点击“专注”后侧栏和次要按钮隐藏，纸面居中变宽。
- 点击“退出专注”后恢复完整编辑页。
- 字数变化能同步到纸面底部和创作侧栏。
- 富文本工具条仍能在选中文本时出现并正常执行格式命令。
- 现有单元测试保持通过。
